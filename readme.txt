Bonsoir!
Pour compiler notre programme utiliser GCC sur Linux de preference.

Explicite.c:

gcc Explicite.c -lm
.\a.out

Ce fichier contient tout les elements de reponse pour repondre a la partie I du projet. 
L'execution de ce programme va automatiquement ploter les resultats.
On peut facilement modifier les points de controles en modifiant "list_t[]" et "list_x[]" dans le main.
Il ne faut pas oublier dans ce cas la de modifier les valeurs de "length" dans "save_instant_T" et "save_position_x".


Implicite.c:

gcc implicite.c -lm
.\a.out

Ce fichier contient tout les elements de reponse pour repondre a la partie II du projet.
